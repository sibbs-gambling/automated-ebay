INTRODUCTION

============


These code samples are the basic eBay Trading API Code samples in JAVA to get you started using the eBay's Trading API.



DESCRIPTION

===========


The code samples comprise of SOAP code samples and XML samples.
There are 2 SOAP samples:
 
- GeteBayOfficialTime 
 
- AddItem
 

And the xml samples are:
 
- GeteBayOfficialTime
 
- GetSuggestedCategories
 
 

Please see the README for the individual samples on how to run the samples.
