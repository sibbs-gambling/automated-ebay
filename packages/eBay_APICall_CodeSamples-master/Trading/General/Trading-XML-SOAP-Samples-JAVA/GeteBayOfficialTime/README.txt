SUMMARY
=======
This example demonstrates the simplest call to the eBay API. It displays the current official eBay time in GMT.

INSTRUCTIONS
============
1. To get sarted with this example, you will need to set the locations of the files needed for this request. 
   These are the private static variables located at the beginning of the code sample. 

2. You will need to enter your keys (Application, Developer and Certificate) into the config.properties file. 
   You will also need to set the URL of the server you wish to use (Sandbox or Production). 
   Finally you will need to set the Token representing the eBay user aking the call.
