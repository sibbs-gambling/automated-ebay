INTRODUCTION
============
These code samples are the basic eBay Trading API Code samples in Python to get you started using the eBay's Trading API.

DESCRIPTION
===========
The code samples comprise of XML samples.
 - GeteBayOfficialTime
 - GetSuggestedCategories
 - AddItem
 - GetItem
 - GetItemShipping
 
 Please see the README for the individual samples on how to run the samples.
