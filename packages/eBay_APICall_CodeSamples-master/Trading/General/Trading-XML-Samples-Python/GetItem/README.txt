SUMMARY
=======
This example allows the user to enter the Id of an item listed on eBay, the item's details are then displayed, such as Title, Price, Number of Bids, End Time, Time Left, etc.

INSTRUCTIONS
============
To get started with this example you must open the config.ini file and edit the variables as follows:
Developer - Your Developer ID
Application - Your Application ID
Certificate - Your Certificate ID
URL - The URL of the eBay server you wish to use (Sandbox or Production)
Token - The Authentication token representing the eBay user who is making the call
