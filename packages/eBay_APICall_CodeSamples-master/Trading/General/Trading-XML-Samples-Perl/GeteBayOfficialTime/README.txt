SUMMARY
========
This example displays the current official eBay time in GMT.

INSTRUCTIONS
============
To get started with this example, you must first enter you sandbox keys into the relevant variables 
in the file ../common/config.pl :
$devID - Developer ID
$appID - Application ID
$certID - Certificate ID

Then, you must set the $requestToken variable to equal the Authentication Token representing the user who is making the call.
